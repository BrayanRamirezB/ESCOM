SOBRECARGA DE FUNCIONES

Sobrecarga de funciones

Definición de sobrecarga. En Java es posible definir dos o más métodos dentro de la misma clase que tengan el mismo nombre, pero con sus listas de parámetros distintas. Cuando ocurre esto, se dice que los modos están sobrecargados y a este proceso se le denomina sobrecarga de modo.


Como hace la selección. Cuando se invoca un método sobrecargado, Java utiliza el tipo y/o el número de argumentos como guía para determinar la versión del método sobrecargado que realmente debe llamar.

       class Estudiante {
         private String nombre;
         private String direccion;
         private double gpa;
         public Estudiante(String nombreEstudiante) {
           nombre = nombreEstudiante;
         }
         public Estudiante(String nombreEstudiante, String direccionEstudiante) {
           nombre = nombreEstudiante;
           direccion = direccionEstudiante;
         }
         public Estudiante() {
         }
         public void setNombre(String nombreEstudiante) {
           nombre = nombreEstudiante;
         }
         public void setDireccion(String direccionEstudiante) {
           direccion = direccionEstudiante;
         }
         public String getNombre() {
           return nombre;
         }
         public String getDireccion() {
           return direccion;
         }
         public double getGpa() {
           return gpa;
         }
         public void crearGPA(Curso nuevoCurso, char grado) {
           // se usa el grado y el curso para actualizar el gpa
         }
       }

       public class EstudianteDriver {
         public static void main(String args[]) {
         Estudiante estudiante = new Estudiante("Jesus", "poli 43");
         System.out.println(estudiante.getNombre());
    
         Estudiante est = new Estudiante();
         System.out.println(est.getNombre());
        }
      }

La salida del programa 1, es:

        Jesus
        null
