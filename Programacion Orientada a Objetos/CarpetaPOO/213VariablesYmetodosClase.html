VARIABLES Y MÉTODOS DE CLASE

Variables

Definición de variable. La variable es la unidad básica de almacenamiento en un programa. Una variable se define mediante la combinación de un identificador, un tipo y un inicializador opcional. Además, todas las variables tienen un ámbito, que define su visibilidad y su tiempo de vida.


Inicialización dinámica. Sólamente se han usado constantes como inicializadores, otra forma de inicializar las variables es la dinámica, donde se utiliza expresiones válidas en el instante en el que la variable es declarada. Como en el ejemplo siguiente:


Programa 1.


        class InicializacionDinamica {
          public static void main(String args[]) {
            double a = 3.0, b = 4.0;
            
            double c = Math.sqrt(a * a + b * b);
            System.out.println("La hipotenusa es " + c);
          }
         }

Programa 2.


        public class CalculoBono {
          public static void main(String[] args) {
            int salario; // salario de la persona
            String bonoImpreso; // especifica horas de trabajo
    
            salario = 50000;
            bonoImpreso = "Bono = $" + (.02 * salario);
            System.out.println(bonoImpreso);
          } // fin main
        } // fin clase CalculoBono

Programa 3.


        public class ConvertidorTemperatura {
          public static void main(String[] args) {
            final double PUNTO_CONGELAMIENTO = 32.0;
            final double FACTOR_CONVERSION = 5.0 / 9.0;
            double fahrenheit = 50; // temperatura en Fahrenheit
            double celsius; // temperatura en Celsius

            celsius = FACTOR_CONVERSION * (fahrenheit - PUNTO_CONGELAMIENTO);
            System.out.println(fahrenheit + " grados Fahrenheit = " +
            celsius + " grados Celsius.");
          } // fin main
        } // fin clase ConvertidorTemperatura

Ámbito de la variable. Las variables se declaran al comienzo del método main(). Pero también se permite que las variables sean declaradas dentro de cualquier bloque. Un bloque define un ámbito. Así, cada vez que empiece un nuevo bloque estamos creando un nuevo ámbito.


        class Ambito {
          public static void main(String args[]) {
            int x;
            x = 10;
            
            if(X == 10) {
              int y = 20;
              
              System.out.println("x e y: " + x + " " + y);
              x = y * 2;
            }
            y = 100;
            System.out.println("x es " + x);
            System.out.println("y es " + y);
           }
         }

El tiempo de vida de las variables. Si la declaración de una variable incluye un inicializador, entonces esa variable será inicializada cada vez que se entra en el bloque en el que está declarada. Como se muestra en el siguiente programa:


        class TiempoVida {
          public static void main(String args[]) {
            int x;

            for(x = 0; x < 3; x++) {
              int y = -1;
              System.out.println("y es: " + y)
              y = 100;
              System.out.println("y es ahora: " + y);
            }
           }
        }

Métodos de clase. Los métodos de clase, como las variables de clase, se relacionan con la clase como un todo y no se relacionan con objetos individuales. Como tal, si necesita realizar una tarea que involucre a la clase como un todo, entonces debe implementar y utilizar un método de clase. Como lo métodos de clase definidos en la clase Java API Math; por ejemplo, Math.round y Math.sqrt. Ahora escribirá sus propios métodos de clase. Métodos de clase a menudo acceden a variables de clase, y al escribir sus propios métodos de clase, tendrá la oportunidad de ver cómo acceder a las variables de clase que ha definido.


       public class Raton4 {
         //variables de clase
         private static int ratonCuenta;
         private static Raton4 ratonJoven; 
         
         private int edad;
  
         public Raton4() {
           Raton4.ratonCuenta++;
           Raton4.ratonJoven = this;
       }
  
       //Metodo de clase
       public static void imprimirRatonCuenta() {
         System.out.println("Total de ratones = " +  Raton4.ratonCuenta);
       }
       public void viejoPorUnDia() {
         this.edad++;
       }

       public static void main(String[] args) {
         Raton4 pinky = new Raton4();
         pinky.viejoPorUnDia();
         Raton4.imprimirRatonCuenta();
       }
     } // fin clase Raton4

Para declarar un método de clase, use esta sintaxis para el encabezado del método:


       private-o-public static tipo-retorno nombre_metodo(parameters)

El prefijo de punto de la variable de referencia es necesario al llamar a una instancia método desde dentro de un método de clase.


Si intenta acceder a un método de instancia directamente desde un método de clase, verá un mensaje de error como el que se presenta a continuación:


       Non-static "Nombre metodo" no puede ser referenciado de un "contexto static"

       Non-static "method-name" cannot be referenced from a static context
