Programación orientada a objetos
Práctica 1

Temario

¿CÓMO COMPILAR PROGRAMAS ORIENTADOS A OBJETOS?

Objetivo

Se compilarán y ejecutarán los programas en java, usando la consola.

Actividades

Compilar y ejecutar el siguiente programa:


Programa UNO:


       public class A {
         String a;
         public String A() {
           return "Hola";
         }
         public void setPalabra(String b) {
           a = b;
         }
         public String getPalabra() {
           return a;
         }
         public void metodo1() {
           System.out.println("Metodo 1");
         }
         public void metodo2() {
           System.out.println("Metodo 2");
         }
         public static void main(String[] args) {
           A obj = new A();
           obj.metodo1();
           obj.metodo2();
           A obj1 = new A();
           System.out.println(obj1.A());
           A obj2 = new A();
           obj2.setPalabra("Mundo");
           System.out.println(obj2.getPalabra());
         }
       }

Cuestionario


Pregunta 1. ¿Cómo se debe llamar el programa UNO? Justificar su respuesta.


Pregunta 2. El programa UNO, ahora dividilo en dos clases. Luego: ¿Cómo se debe llamar el programa? Entregar el código.




Sugerencias

Sugerencia 1. Para compilar el programa en la consola, usar el comando: javac ProgramaUNO.java. Para ejecutarlo, usar el comando: java ProgramaUNO.


Sugerencia 2. Dividir el programaUNO en dos, considerando dos clases. La clase principal se declara como public y la otra no.
