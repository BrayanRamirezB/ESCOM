ATRIBUTOS Y MÉTODOS

Atributos

Definición de atributos. Un atributo es una propiedad que se asocia con un objeto: esto sirve para describir al objeto y mantener algún valor que es requerido para el procesamiento.


Definición de campo de datos. Un campo es una variable definida directamente dentro de una clase y corresponde a un atributo. Cada instancia del objeto tendrá almacenamiento para el campo.

El mecanismo de clases en los lenguajes orientados a objetos proporciona una forma de crear tales objetos. Una clase es un diseño que puede reutilizarse tantas veces como desee para crear objetos. Por ejemplo, considere un sistema orientado a objetos para una universidad. Hay objetos de estudiantes, objetos de instructor, objetos de miembros del personal, etc. Antes de que se creen tales objetos, creamos clases que sirven como planos para estudiantes, instructores, miembros del personal y cursos de la siguiente manera:



Código en java. El siguiente código es un ejemplo del uso de atributos:


 public class Estudiante {
   // codigo para implemtar un estudiante
 }
 public class Instructor {
   // codigo para implementar un instructor
 }
 public class MiembroPersonal {
   // codigo para implementar un miembro del personal
 }
 public class Curso {
   // codigo para implementar un curso
 }


Antes de ver cómo poner los detalles de la clase, veamos cómo crear objetos. usando estas clases. El proceso de creación de un objeto también se denomina instanciación. Cada class introduce un nuevo nombre de tipo. Por lo tanto, estudiante, instructor, miembro del personal y los cursos son los tipos que hemos introducido.



Código en java. El siguiente código instancia un nuevo objeto del tipo Estudiante:


 new Estudiante();

Código en java. El siguiente código crea una variable del tipo Estudiante:


 Estudiante Jorge;

Métodos

Definición de método. El término función se ha utilizado para describir una subrutina con nombre. El término que se ha usado más frecuentemente en Java es método, al ser "una manera de hacer algo". Si se desea, es posible seguir pensando en funciones. Verdaderamente sólo hay una diferencia sintáctica, pero de ahora en adelante se usará el término "método" en lugar del término "función". Los métodos en Java determinan los mensajes que puede recibir un objeto. En esta sección se aprenderá lo simple que es definir un método. Las partes fundamentales de un método son su nombre, sus parámetros, el tipo de retorno y el cuerpo. Hay dos tipos básicos de métodos, métodos de instancia y métodos de clase.


Código en java. Su forma básica, es:


     tipoRetorno nombreMetodo(lista de parámetros) {
       / * Cuerpo del método * /
     }

Piensa en Java, Bruce Eckel, Pág. 64


Definición de métodos de instancia. Son aquellos métodos que están asociados con una instancia particular de una clase. Por ejemplo, para llamar al método length de la clase String, debe asociarlo con una cadena en particular.


Código en java. En el siguiente ejemplo, observe cómo la cadena primerNombre está asociada con el método length:


     tamanioPrimerNombre = primerNombre.length();

Definición de método de clase. Los métodos de clase son métodos que están asociados con una clase completa, no con una instancia particular de un clase. Por ejemplo, hay una clase Math que contiene muchos métodos de clase. Sus métodos están asociados con matemáticas en general, no con una instancia particular de matemáticas (una instancia particular de matemáticas ni siquiera hace sentido). Para llamar a un método de clase, prefije el nombre del método con el nombre de la clase que lo define. Por ejemplo, la clase Math contiene un método round que devuelve la versión redondeada de un valor dado. Llamar el método round, lo prefijas con Math así:


     paymentInDollars = Math.round(calcularGanancias);

Introduction to programming with Java: A problem solving Approach, John Dean and Raymond Dean, Pág. 152

Código en java. El siguiente código es un forma de sumar números sin usar métodos:


 int sum = 0;
 for (int i = 1; i <= 10; i++)
 sum += i;
 System.out.println("Sum from 1 to 10 is " + sum);
 sum = 0;
 for (int i = 20; i <= 30; i++)
 sum += i;
 System.out.println("Sum from 20 to 30 is " + sum);
 sum = 0;
 for (int i = 35; i <= 45; i++)
 sum += i;
 System.out.println("Sum from 35 to 45 is " + sum);

Código en java. El siguiente código es la forma de sumar números usando métodos:


 public static int sum(int i1, int i2) {
   int sum = 0;
   for (int i = i1; i <= i2; i++)
   sum += i;
   return sum;
 }

 public static void main(String[] args) {
   System.out.println("Sum from 1 to 10 is " + sum(1, 10));
   System.out.println("Sum from 20 to 30 is " + sum(20, 30));
   System.out.println("Sum from 35 to 45 is " + sum(35, 45));
 }

Introduction to Java programing, Y. Daniel Liang, Pág. 156


Código en java. El siguiente código es la clase Estudiante:


       public class Estudiante {
         private String nombre;
         public void setNombre(String nombreEstudiante) {
           nombre = nombreEstudiante;
         }
         public String getNombre() {
           return nombre;
         }
       }

Código en java. El siguiente código crea el objeto aEstudiante con el uso de la clase Estudiante:


      Estudiante aEstudiante = new Estudiante();
      aEstudiante.setNombre("Roberto");

Código en java. El siguiente código es el método principal de la clase Estudiante:


      public static void main(String[] s) {
        Estudiante estudiante1 = new Estudiante();
        Estudiante estudiante2 = new Estudiante();
        estudiante1.setNombre("Juan");
        estudiante2.setNombre("Maria");
        System.out.println(estudiante1.getNombre());
        System.out.println(estudiante2.getNombre());
      }
